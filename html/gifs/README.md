# Bugchess Piece Animation GIFs

This folder contains animated GIFs showing the creation (merge) animation for each special piece type in Bugchess. Each GIF is a tightly-cropped, high-quality capture of the in-game merge event for that piece.

## Files

- `horizontal-bar-merge.gif` — Horizontal Bar: 3 basic pieces in a row merge into a Bar.
- `vertical-bar-merge.gif` — Vertical Bar: 3 basic pieces in a column merge into a Bar.
- `cross-merge.gif` — Cross: 5 basic pieces in a cross pattern merge into a Cross.
- `sniper-merge.gif` — Sniper: 5 basic pieces in an X pattern merge into a Sniper.
- `castle-merge.gif` — Castle: 4 basic pieces in a rotated square merge into a Castle.
- `queen-merge.gif` — Queen: 8 basic pieces in a diamond pattern merge into a Queen.

Each GIF shows only the merge animation for that piece type, with no extra UI or distractions. These are used for the landing page and documentation.
